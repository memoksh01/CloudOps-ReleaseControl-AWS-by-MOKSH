# CloudOps-ReleaseControl-AWS

Automated release governance and environment compliance framework built on AWS.

## Key Features
- Multi-environment consistency (Dev, QA, UAT, Prod)
- Automated quality gates
- Pre-release validation
- Manual approvals
- Audit logging

## Tools
- AWS CodePipeline
- AWS CodeBuild
- Terraform
- SonarCloud

## Outcome
Ensured controlled, compliant, and quality-driven production releases.
